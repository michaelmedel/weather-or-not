# Year-Ago Weather

What was the weather like a year ago today?

Stop arguing with your friends — prove it.

https://yearagoweather.com

---

## Overview

Year-Ago Weather lets you enter any city and instantly compare today’s weather with the same day last year. It focuses on simplicity, speed, and clarity.

---

## Features

- Compare today vs. last year’s weather  
- Displays high and low temperatures  
- Includes weather conditions (sunny, rain, snow, etc.)  
- Handles duplicate city names intelligently  
- Mobile-first, clean interface  
- No login or setup required  

---

## Tech Stack

- HTML, CSS, JavaScript (no frameworks)  
- Open-Meteo API (no API key required)  
- GitHub Pages for hosting  
- Namecheap for domain management  

---

## How It Works

1. User enters a city  
2. The app geocodes the location to latitude and longitude  
3. Fetches:
   - Current forecast data  
   - Historical data for the same date last year  
4. Compares and displays the results
